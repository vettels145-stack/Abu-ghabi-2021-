# Друзья! Это приложение называется "Битва до победы!" 
## Правила игры:
1. Вы играете против "компьютер" под названием LH 44.
    1. Наша задача - выиграть
3. Проигрывает тот, кто заберет последнюю спичку из общего количества.
4. Первый всегда ходит человек, только если он не напишет вместо "yes" или "да" что-то другое, игра такое не прощает и передает ход сопернику.)
5. Брать можно от 1 до 3 спичек за раз, ни больше, ни меньше. При любой отклонении от правил игра отправляет запрос еще раз.
